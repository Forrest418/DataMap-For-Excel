DataMap For Excel，数据地图
======
DataMap，是一款便捷的Microsoft Excel地图应用插件和数据可视化工具，可以让用户在Excel中轻松制作出专业的地图和图表，展示数据的空间分布和关系，提高数据分析的效率和质量。
